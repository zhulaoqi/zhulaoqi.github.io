六、常见问题

Q: 部署后线上没更新？
A: 检查两件事：① _config.yml 的 deploy.branch 是否与 GitHub 仓库默认分支一致；② GitHub Pages 有 1-2 分钟缓存，等一等或强制刷新（Ctrl+Shift+R）。

Q: 本地预览正常但部署后样式丢失？
A: 检查 _config.yml 中的 url 和 root 是否正确配置了你的 GitHub Pages 地址。

Q: 怎么删除一篇文章？
A: 直接删除 source/_posts/ 下对应的 .md 文件，然后重新部署。

Q: 想改博客样式？
A: 编辑 source/_data/styles.styl，不要直接改 themes/next/ 里的文件。

保持写作，保持输出。最好的笔记系统是你真正在用的那一个。
